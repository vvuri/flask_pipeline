## Django simple project

### Steps
- Install Django 
  - ```$ pip install django```
  - ```$ django-admin version``` -> 4.0.1
- Create project:
  - ```$ django-admin startproject django_first```
  - ```$ cd django_first```
- Run dev server: 
  - ```$ python .\manage.py runserver```
  - Starting development server at http://127.0.0.1:8000/
